About Sarlack-Lab

I built the Sarlack to "devour malware in a sandbox". The server automatically grabbed and analyzed malware samples for personal research and to assist my SOC. While studying network detection trends, I began to notice patterns among malicious IP addresses. Many of the IPs appeared to be hosted in similar IP spaces (even across different ASNs). I created a map to visualize this threat landscape using the fantastic resources provided by Xylitol, abuse.ch, Hybrid Analysis, Paul Melson, AZORult Tracker, Fred HK, Dee, and Honeypot.FYI.

In the future, I plan on training the Sarlack to autonomously hunt C2 servers by looking for patterns among sandbox reports. Once the C2 protocol has been verified and multiple samples have been analyzed, the C2 server will be shared publicly and added to the map.
